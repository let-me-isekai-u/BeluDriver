README - Tiếng Việt

Thư mục này chứa model nghiệp vụ chính của tài xế.

Ý nghĩa các file:
- `broker_rides_model.dart`: model danh sách đơn đã đẩy / đã chia sẻ.
- `driver_profile_model.dart`: model hồ sơ tài xế.
- `ride_detail_model.dart`: model chi tiết chuyến xe.
- `ride_model.dart`: model item chuyến xe trong danh sách.
- `withdrawal_history_model.dart`: model lịch sử rút tiền.

README - English

This folder contains core business models for the driver side.

File purposes:
- `broker_rides_model.dart`: list model for pushed/shared rides.
- `driver_profile_model.dart`: driver profile model.
- `ride_detail_model.dart`: ride detail model.
- `ride_model.dart`: ride item model used in lists.
- `withdrawal_history_model.dart`: withdrawal history model.
