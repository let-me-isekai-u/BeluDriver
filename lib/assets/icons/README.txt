README - Tiếng Việt

Thư mục này chứa icon và logo dùng trong app.

Ý nghĩa các file:
- `ActivityLogo.png`: icon hoặc hình phụ trợ cho màn hình hoạt động.
- `BeluCar_logo.jpg`: logo BeluCar.
- `BeluDriver_launcher.png`: icon launcher hoặc branding BeluDriver.
- `dong_duong_driver_logo.png`: logo Đồng Dương Driver.
- `icons8-zalo-100.png`: icon Zalo cho chức năng liên hệ hỗ trợ.
- `united-states.png`: hình đại diện cho ngôn ngữ tiếng Anh.
- `vietnam.png`: hình đại diện cho ngôn ngữ tiếng Việt.

Lưu ý:
- Nếu thay đổi logo hoặc icon cần kiểm tra lại tất cả chỗ đang gọi `Image.asset`.

README - English

This folder contains icons and logos used by the app.

File purposes:
- `ActivityLogo.png`: support icon for the activity screen.
- `BeluCar_logo.jpg`: BeluCar logo.
- `BeluDriver_launcher.png`: launcher icon or BeluDriver branding asset.
- `dong_duong_driver_logo.png`: Dong Duong Driver logo.
- `icons8-zalo-100.png`: Zalo icon used in support/contact flows.
- `united-states.png`: English language indicator image.
- `vietnam.png`: Vietnamese language indicator image.

Note:
- If any logo or icon is changed, review all `Image.asset` usages.
