README - Tiếng Việt

Thư mục này chứa màn hình chat liên quan đến đơn đẩy.

Ý nghĩa các file:
- `chat_group_list_screen.dart`: danh sách các nhóm chat tài xế.
- `chat_screen.dart`: màn hình chat của một nhóm, có gửi tin nhắn và thao tác liên quan đến đơn.

README - English

This folder contains chat screens related to pushed rides.

File purposes:
- `chat_group_list_screen.dart`: list of driver chat groups.
- `chat_screen.dart`: chat screen for a group, including messaging and ride-related actions.
