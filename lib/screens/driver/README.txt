README - Tiếng Việt

Thư mục này chứa các màn hình nghiệp vụ chính của tài xế.

Ý nghĩa các file:
- `activity_history.dart`: màn hình hoạt động, gồm đang diễn ra, lịch sử, đơn đã đẩy.
- `driver_booking_confirm.dart`: xác nhận thông tin trước khi tạo / đẩy đơn.
- `driver_booking_screen.dart`: form nhập thông tin để đẩy đơn.
- `driver_home.dart`: màn hình home chính của tài xế.
- `driver_login_screen.dart`: màn hình đăng nhập.
- `driver_profile.dart`: hồ sơ tài xế, tài chính, trạng thái tài khoản.
- `driver_register_screen.dart`: màn hình đăng ký tài xế.
- `driver_update_profile_screen.dart`: cập nhật thông tin hồ sơ.
- `recieve_order_screen.dart`: màn hình nhận đơn đang chờ.
- `ride_detail_screen.dart`: màn hình chi tiết một chuyến xe.
- `wallet_history_screen.dart`: lịch sử biến động ví.
- `withdrawal_history_screen.dart`: lịch sử rút tiền.

Lưu ý:
- Nhóm file này là trung tâm nghiệp vụ của app.
- Một số màn hình đã được tách provider riêng trong `providers/driver`.

README - English

This folder contains the main business screens for the driver workflow.

File purposes:
- `activity_history.dart`: activity screen with ongoing rides, history, and pushed rides.
- `driver_booking_confirm.dart`: confirmation screen before creating/pushing a ride.
- `driver_booking_screen.dart`: form screen for pushing a ride.
- `driver_home.dart`: main driver home screen.
- `driver_login_screen.dart`: login screen.
- `driver_profile.dart`: driver profile, finance, and account status screen.
- `driver_register_screen.dart`: driver registration screen.
- `driver_update_profile_screen.dart`: profile update screen.
- `recieve_order_screen.dart`: screen for receiving waiting rides.
- `ride_detail_screen.dart`: detailed ride screen.
- `wallet_history_screen.dart`: wallet transaction history screen.
- `withdrawal_history_screen.dart`: withdrawal history screen.

Notes:
- This folder is the core business center of the app.
- Some screens already have dedicated providers under `providers/driver`.
