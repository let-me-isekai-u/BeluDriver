README - Tiếng Việt

Thư mục này chứa các màn hình chụp và xử lý ảnh khuôn mặt.

Ý nghĩa các file:
- `face_capture_mode.dart`: định nghĩa mode chụp mặt.
- `front_face_capture_screen.dart`: chụp mặt chính diện.
- `left_face_capture_screen.dart`: chụp nghiêng trái.
- `right_face_capture_screen.dart`: chụp nghiêng phải.
- `single_face_capture_screen.dart`: màn hình gốc xử lý camera / chụp một bước.

Lưu ý:
- Đây là nhóm có logic camera nhiều, cần test kỹ nếu sửa.

README - English

This folder contains face capture and face-image processing screens.

File purposes:
- `face_capture_mode.dart`: defines face capture modes.
- `front_face_capture_screen.dart`: front-face capture screen.
- `left_face_capture_screen.dart`: left-side capture screen.
- `right_face_capture_screen.dart`: right-side capture screen.
- `single_face_capture_screen.dart`: base screen handling camera and single-step capture logic.

Note:
- This group contains a lot of camera logic and should be tested carefully after changes.
