README - Tiếng Việt

Thư mục này chứa popup / màn hình cho luồng KYC.

Ý nghĩa các file:
- `kyc_deposit_requirement_popup.dart`: popup thông báo về yêu cầu ký quỹ / số dư.
- `kyc_popup.dart`: popup tổng hợp các bước KYC.
- `kyc_step_citizen_card.dart`: bước chụp CCCD hai mặt.
- `kyc_step_driver_license.dart`: bước chụp bằng lái.
- `kyc_step_face_verification.dart`: bước xác thực khuôn mặt.
- `kyc_step_portrait.dart`: bước chụp chân dung.
- `kyc_step_vehicle_photo.dart`: bước chụp ảnh xe.
- `kyc_step_vehicle_registration.dart`: bước chụp đăng ký xe.
- `kyc_submit_success_popup.dart`: popup gửi KYC thành công.
- `resubmit_kyc_notification.dart`: thông báo hồ sơ KYC cần bổ sung.

README - English

This folder contains popup/screens for the KYC flow.

File purposes:
- `kyc_deposit_requirement_popup.dart`: popup about deposit/balance requirements.
- `kyc_popup.dart`: main popup that orchestrates KYC steps.
- `kyc_step_citizen_card.dart`: citizen ID capture step.
- `kyc_step_driver_license.dart`: driver license capture step.
- `kyc_step_face_verification.dart`: face verification step.
- `kyc_step_portrait.dart`: portrait capture step.
- `kyc_step_vehicle_photo.dart`: vehicle photo step.
- `kyc_step_vehicle_registration.dart`: vehicle registration capture step.
- `kyc_submit_success_popup.dart`: success popup after KYC submission.
- `resubmit_kyc_notification.dart`: notification that KYC needs resubmission.
