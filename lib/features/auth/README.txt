README - Tiếng Việt

Thư mục này chứa điểm vào theo feature cho xác thực.

File hiện có:
- `auth.dart`: file barrel export các thành phần auth như login, register, forgot password, splash, terms.

Cách dùng:
- Màn hình mới liên quan auth có thể import một dòng từ file này thay vì import lẻ từng file.

README - English

This folder contains the feature entry point for authentication.

Current file:
- `auth.dart`: barrel file exporting auth-related pieces such as login, registration, forgot password, splash, and terms.

Usage:
- New auth-related files can import this single file instead of many separate imports.
