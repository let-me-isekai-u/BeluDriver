README - Tiếng Việt

Thư mục này chứa điểm vào theo feature cho hồ sơ tài xế.

File hiện có:
- `profile.dart`: barrel export `driver_profile_model`, `profile_provider`, `update_profile_provider` và các màn hình profile.

Cách dùng:
- File mới liên quan profile có thể import từ đây để dùng model / provider / screen mà không cần nhiều dòng import lẻ.

README - English

This folder contains the feature entry point for driver profile.

Current file:
- `profile.dart`: barrel exporting `driver_profile_model`, `profile_provider`, `update_profile_provider`, and profile screens.

Usage:
- New profile-related files can import this file to access models/providers/screens without many separate imports.
