README - Tiếng Việt

Thư mục này là nhóm feature của tài xế, được tách tiếp thành các nhánh con.

Thư mục con:
- `activity/`: hoạt động chuyến xe và đơn đã đẩy.
- `finance/`: wallet, rút tiền, nạp tiền.
- `profile/`: hồ sơ tài xế.
- `rides/`: chi tiết chuyến xe và model liên quan.

Lưu ý:
- Thư mục này hiện chưa có file barrel tổng hợp cấp `driver/`, chủ yếu dùng các barrel ở nhánh con.

README - English

This folder groups driver-related features and is split into smaller branches.

Subfolders:
- `activity/`: ride activity and pushed rides.
- `finance/`: wallet, withdrawal, and deposit flows.
- `profile/`: driver profile.
- `rides/`: ride detail and related models.

Note:
- There is currently no single top-level `driver` barrel file; the sub-feature barrels are used directly.
