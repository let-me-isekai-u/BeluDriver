README - Tiếng Việt

Thư mục này chứa provider riêng cho các màn hình driver đã được tách logic.

Ý nghĩa các file:
- `activity_history_provider.dart`: dữ liệu và thao tác cho màn hình hoạt động.
- `deposit_provider.dart`: tạo / hủy yêu cầu nạp tiền.
- `login_provider.dart`: xử lý đăng nhập tài xế.
- `profile_provider.dart`: tải hồ sơ tài xế, logout, delete account, support.
- `register_provider.dart`: xử lý đăng ký tài xế.
- `ride_detail_provider.dart`: tải chi tiết chuyến và cập nhật trạng thái chuyến.
- `update_profile_provider.dart`: cập nhật hồ sơ tài xế và ảnh đại diện.
- `wallet_history_provider.dart`: tải lịch sử tài chính / số dư ví.
- `withdrawal_history_provider.dart`: tải lịch sử rút tiền.

README - English

This folder contains dedicated providers for driver screens whose logic has been extracted.

File purposes:
- `activity_history_provider.dart`: data and actions for the activity screen.
- `deposit_provider.dart`: create/cancel deposit requests.
- `login_provider.dart`: driver login logic.
- `profile_provider.dart`: driver profile loading, logout, delete account, and support actions.
- `register_provider.dart`: driver registration logic.
- `ride_detail_provider.dart`: ride detail loading and ride status updates.
- `update_profile_provider.dart`: profile and avatar update logic.
- `wallet_history_provider.dart`: wallet balance and financial history loading.
- `withdrawal_history_provider.dart`: withdrawal history loading.
